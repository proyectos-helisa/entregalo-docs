# Dashboard

El Dashboard es la pantalla principal de Entrégalo, diseñada para proporcionar una visión general y rápida del estado de las operaciones de transporte.

### Información que muestra

1. Resumen de remesas (pendientes, registradas, completadas, canceladas)
2. Estado de viajes/manifiestos
3. Gráficos de rendimiento (por ejemplo, viajes completados por mes)
4. Alertas y notificaciones importantes
5. Indicadores clave de desempeño (KPIs) relevantes para el transporte de carga

### Cómo se usa

1. Al iniciar sesión, el usuario es dirigido automáticamente al Dashboard.
2. Los diferentes widgets y gráficos son interactivos, permitiendo al usuario hacer clic para obtener más detalles.
3. El usuario puede personalizar la vista del Dashboard según sus preferencias y responsabilidades.
4. Desde aquí, se puede acceder rápidamente a los diferentes módulos de la aplicación mediante un menú de navegación.

El Dashboard sirve como punto de partida para la toma de decisiones diarias y la planificación estratégica, ofreciendo una visión clara y concisa del estado actual de las operaciones de transporte.
